About Facial Expression Recognition

This application uses a Convolutional Neural Network (CNN) trained on the FER2013 dataset to recognize facial expressions.

The model can detect 7 different emotions:

  • Angry - Expressions showing anger or irritation
  • Disgust - Expressions showing aversion or repulsion
  • Fear - Expressions showing fear or anxiety
  • Happy - Expressions showing joy or pleasure
  • Neutral - Expressions showing no particular emotion
  • Sad - Expressions showing sadness or unhappiness
  • Surprise - Expressions showing astonishment or shock

The model was trained using TensorFlow/Keras and is served using a Flask backend.
